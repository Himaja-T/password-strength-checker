# Password Strength Checker

## Description
This is a Python-based password strength checker that evaluates the security level of a password based on multiple criteria such as length, character variety, and use of special symbols.

## Features
- Checks password length (minimum 8 characters)
- Detects uppercase and lowercase letters
- Identifies numeric digits
- Validates special characters
- Classifies password as Weak, Medium, or Strong

## Technologies Used
- Python
- Regular Expressions (re module)

##  How to Run
1. Clone the repository:
   git clone https://github.com/your-username/password-strength-checker.git

2. Navigate to the folder:
   cd password-strength-checker

3. Run the program:
   python main.py

## Example Output
Enter password: Test@123  
Strength: Strong
